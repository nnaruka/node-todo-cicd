Create and Login to EC2 instance
Ensure I'm able to connect from my IP.

Ubuntu
============
1. Generate ssh key and upload into github 
    ssh-keygen -t ed25519 -C "github-key"
    cat ~/.ssh/id_ed25519.pub
    Upload to github -> settings -> SSH and GPG Keys

2. git clone git@github.com:nnaruka/node-todo-cicd.git

3. Docker
    sudo apt install docker.io
    sudo usermod -aG docker $USER
    sudo su - ubuntu
    docker ps

4. Install AWS Cli
    curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
    sudo apt install unzip
    sudo unzip awscliv2.zip
    sudo ./aws/install
    aws --version

5. Configure aws login from EC2 to ECR so it can connect
    Create IAM User bhide
    User bhide -> Security Credentials -> create Access key -> aws cli

    EC2
    aws configure
    setup Access key and Secret Access key from dashboard
    

ECR
=============
1. Create Repository in ECR "node-app"
2. Click node-app-> images -> push command (view them)
    
    aws ecr get-login-password --region eu-north-1 | docker login --username AWS --password-stdin 333519902580.dkr.ecr.eu-north-1.amazonaws.com
    docker build -t node-app .
    docker tag node-app:latest 333519902580.dkr.ecr.eu-north-1.amazonaws.com/node-app:latest
    docker push 333519902580.dkr.ecr.eu-north-1.amazonaws.com/node-app:latest
